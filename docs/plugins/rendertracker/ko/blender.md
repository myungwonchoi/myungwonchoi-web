# Blender 애드온

설치 파일에서 Blender 버전을 체크하여 설치
Preferences > Add-ons에서 RenderTracker (Blender)를 활성화하여 RenderTracker와 연동합니다.

상단 Render 메뉴의 RenderTracker...에서 앱을 실행할 수 있습니다.

블렌더에서는 F12로 현재 프레임을 렌더할 때 RenderTracker 앱으로 알림이 오는 현상이 생길 수도 있습니다.
(첫 프레임에서 렌더하면 해당 현상이 나타납니다.)  
이 경우, Preferences > Add-ons에서 RenderTracker (Blender)에서 Detect Rendering after 1st frame (Exclude Still Renders)를 활성화하면 해당 현상이 해결됩니다.  
하지만 해당 옵션을 활성화하면 첫 프레임이 렌더 된 이후에야 RenderTracker에 알림이 오기 시작합니다.

블렌더에서는 RenderTracker 앱으로 텍스트로 된 가벼운 렌더링 정보만 전송하기 떄문에 실질적인 렌더링 속도에는 영향이 거의 없습니다.
