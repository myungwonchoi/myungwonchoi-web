# Cinema 4D 플러그인

설치 파일에서 Cinema 4D 버전을 체크하여 설치하면 별도 세팅없이 자동으로 RenderTracker와 연동됨.

Extensions 메뉴에서 RenderTracker 앱을 실행할 수 있습니다.

Cinema 4D Render Setting-Save가 활성화되어야지만 RenderTracker에서 썸네일이 표시됨.
(현재 버전은 옥테인이나 레드쉬프트의 AOV 세부 경로를 추적하지 못함.)

시포디에서는 RenderTracker 앱으로 텍스트 정보만 신호를 보내기 떄문에 실질적인 렌더링 속도에는 영향이 거의 없습니다.
